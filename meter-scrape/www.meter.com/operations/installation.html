Meter’s implementation program methodology

Meter Network deployments occur in 3 stages: Discovery, Planning, and Execution, culminating in Ongoing Management.

Meter defines a Program (company-wide implementation) in all deployments, enabling streamlined execution of all Projects (site-level networks).
